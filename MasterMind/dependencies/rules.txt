Hello,Welcome to the game of mastermind.
This is only a brief tutorial on how this
game is to be played.

MasterMind is a game involving 2 players.
In this game,one person makes up a combination
of 4 coloured balls.The second player must go 
through turns and try to figure out what combination
the first player has chosen.

Here,the computer chooses and the player must guess
within the specified number of tries.

To ease the guessing,the player is provided with pins
for each guess:
1.Grey Pin: symbolizes no pin

2.Red Pin:the player has guessed one colour correctly
along with its correct position.

3.White Pin:the player has guessed one colour correctly
but not gotten its position right.

The game shows only the most recent 5 guesses.
