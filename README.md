# MasterMind
This is my version of the classic board game mastermind.

## Requirements:<br>
* Python 3 or above

## How to Play:<br>

### Open terminal on your PC and type in the following commands:<br> 
* git clone https://github.com/never2average/MasterMind.git
* cd MasterMind/MasterMind
* python3 __init__.py

### Enjoy the game!!
